# 车辆管理系统

## 项目简介

车辆管理系统是一个面向企业车辆资产管理的前端原型系统，涵盖车辆全生命周期管理，包括车辆建档、资产管理和车型管理三大核心模块。系统采用纯 HTML/CSS/JavaScript 实现，可作为前端高保真原型或后续前后端分离开发的基础模板。

## 功能模块

### 1. 车辆建档
- **列表页**：支持多维度查询筛选（车架号、车牌号、车型名称、车辆状态等），分页展示车辆建档数据，支持批量导入、单条录入、数据导出。
- **新增页**：完整的新车信息录入表单，涵盖车辆基础信息、归属信息、车辆照片、行驶证信息、车型信息、车辆功能配置及备注。
- **详情页**：车辆档案详情展示（预留入口，联动资产管理模块）。

### 2. 车辆资产管理
- **列表页**：车辆资产全量列表，支持车辆分配、编辑、删除、导出等操作，多条件组合查询。
- **详情页**：资产详情展示，包含基础信息、调拨信息、变更日志等 Tab 页签，支持查看态与编辑态切换。

### 3. 车型管理
- **列表页**：车型主数据管理，支持可折叠展开的版本子列表，涵盖详情、编辑、新建版本、删除等操作。
- **新增页**：新建车型基础信息及首个版本信息。
- **详情页**：车型及版本信息详情查看与编辑。
- **新建版本页**：基于已有版本快速新建车型版本，继承原有参数并支持修改。

## 使用方式

1. 克隆或下载本项目到本地。
2. 进入 `03-高保真页面` 目录。
3. 打开 `index.html` 或直接打开任意页面文件，即可在浏览器中预览。

> 说明：本系统为纯前端静态页面，数据为本地模拟数据，无需后端服务即可运行预览。

## 技术说明

- **技术栈**：HTML5 + CSS3 + 原生 JavaScript（无框架依赖）。
- **布局方案**：左侧固定侧边栏 + 右侧主内容区，响应式内容区域。
- **公共组件**：`common.css` 提供统一的 CSS 变量、布局框架、按钮、表格、分页、弹窗等样式；`common.js` 提供侧边栏、顶部面包屑、分页组件的初始化与渲染逻辑。
- **浏览器兼容**：推荐 Chrome / Edge 等现代浏览器。

## 注意事项

- `06-备份` 和 `07-bugs` 文件夹已配置在 `.gitignore` 中，不会被提交到 Git 仓库。
- 页面间的跳转链接采用相对路径，确保在本地文件系统或静态服务器中均可正常工作。
